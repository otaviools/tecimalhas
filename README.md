# Tecimalhas 🧵

Uma *landing page* moderna e responsiva desenvolvida para a **Tecimalhas**, uma loja especializada no comércio de tecidos e malhas sediada em Uberlândia, MG. O objetivo deste projeto é fornecer uma presença digital sólida, permitindo aos clientes explorar o catálogo de produtos e encontrar facilmente os canais de contato da empresa.

## 🚀 Demonstração

Você pode visualizar o projeto em execução através do link:  
[https://otaviools.github.io/tecimalhas/](https://otaviools.github.io/tecimalhas/)

## ✨ Funcionalidades

* **Design Responsivo:** Adaptado para dispositivos móveis, tablets e computadores.
* **Catálogo de Produtos:** Seção dedicada à exibição dos tipos de tecidos e malhas disponíveis.
* **Chamadas para Ação (CTA):** Botões estratégicos para contato direto via WhatsApp.
* **Informações de Localização:** Endereço detalhado da loja física em Uberlândia.
* **Interface Intuitiva:** Navegação fluida e foco na experiência do usuário.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estrutura semântica do conteúdo.
* **CSS3:** Estilização, layout (Flexbox/Grid) e animações.
* **JavaScript:** Interatividade e manipulação do DOM.
* **Google Fonts:** Tipografia personalizada para uma identidade visual única.
